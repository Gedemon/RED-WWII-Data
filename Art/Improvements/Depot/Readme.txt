1. Include all .dds, .gr2 and .fxsxml files and set VFS to true for all of them.
2. 'under construction' graphics are included
3. Use the included .png file without 'icon' in its name as the 'unit action' graphics (resize, export as .dds, sizes 64 and 45)
4. You can change the name of the gr2 and fxsxml files, but also change the gr2 name in the fxsxml files. You should not change the dds file(s) name(s).
